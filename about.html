About Resume Maker

What is Resume Maker

Resume maker is a great way for generating your resume within seconds without using any text editor.
Resume Maker generates resume for every professional job type.
You just have to fill the form and submit it your professional resume will be created for you.

Minimum system configurations and requirements.

This can be run in devices which means that you don't need any heavy system configurations.

System RAM : 4 GB
Operating System : Windows 7 or higher (Windows).
Web Browser : Any web browser with recent version.
System Storage : 10MB of storage is enough for saving your RESUME.PDF file

How it works?

We have first created the main page of website with logo and "Get Started" button.
After that we have several links like About, Help, Develops though which user can explore diffrent corners of our website.
When user clicks "Get Started" button he is moved to a form which is a PHP page where he has to fill several entries of form. The entries are made in HTML.
After filling the form user clicks the "CREATE" button which submit the form to a next PHP page which is the main RESUME page.
The data is stored in database and retrieved in RESUME page using MySQL.
The data is represented on the RESUME page in table with 0px border.
Now user can easily print the resume using Print feature of the web browser.
The whole website is designed using CSS.
